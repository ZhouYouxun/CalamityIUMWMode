# 西格纳斯系武器攻击方式

> 目标：每一把武器都转写成一个 Boss 招式概念，只写方向、追踪、弹射、封路和阶段定位，不写具体数值。

---

## P1：暗影使者掉落

宇宙苦无和宇宙灯。

| 招式 | 武器来源 | Boss 版设计 |
|------|----------|-------------|
| 宇宙苦无·CosmicKunai | 宇宙苦无 / Cosmic Kunai | 西格纳斯展开宇宙苦无星阵，星点先慢速定位成图案，随后按图案顺序发射星线，留下清晰但紧凑的穿行缝。 |
| 宇宙灯·Cosmilamp | 宇宙灯 / Cosmilamp | 西格纳斯召出宇宙灯副核心，副核心不追玩家，只周期性封住一片区域；Boss 本体趁空档做轻冲刺。 |

## P2：扭曲虚空武器库

扭曲、暗杀、星座、磁能和天体武器。

| 招式 | 武器来源 | Boss 版设计 |
|------|----------|-------------|
| 以太低语·AethersWhisper | 以太低语 / Aether's Whisper | 西格纳斯展开以太低语星阵，星点先慢速定位成图案，随后按图案顺序发射星线，留下清晰但紧凑的穿行缝。 |
| 死亡升华·DeathsAscension | 死亡升华 / Death's Ascension | 西格纳斯把死亡升华改成有预警的近身斩线：先在玩家移动方向亮出刀痕，随后本体或幻影沿线横切。 |
| 至天飞刀·EmpyreanKnives | 至天飞刀 / Empyrean Knives | 西格纳斯把至天飞刀改成预警斩击：斩线先在玩家前方亮起，短延迟后高速切过，末端再炸出一小圈碎刃。 |
| 星座之王·天龙·KingConstellations | 星座之王·天龙 / King of Constellations, Tenryu | 西格纳斯展开星座之王·天龙星阵，星点先慢速定位成图案，随后按图案顺序发射星线，留下清晰但紧凑的穿行缝。 |
| 磁能熔毁·MagneticMeltdown | 磁能熔毁 / Magnetic Meltdown | 西格纳斯召出磁能熔毁法术核心，核心先漂移锁区，再释放有间隔的光束/魔弹。 |
| 天底·Nadir | 天底 / Nadir | 西格纳斯把天底改成有预警的近身斩线：先在玩家移动方向亮出刀痕，随后本体或幻影沿线横切。 |
| 七发打击者·SevensStriker | 七发打击者 / The Sevens Striker | 西格纳斯以七发打击者为核心放出电弧链，电弧先连接数个预警点，再按连接顺序跳跃放电，要求玩家记住线路。 |
| 金星三叉戟·VenusianTrident | 金星三叉戟 / Venusian Trident | 西格纳斯把金星三叉戟改成预警斩击：斩线先在玩家前方亮起，短延迟后高速切过，末端再炸出一小圈碎刃。 |
| 四季银河·FourSeasons | 四季银河 / Four Seasons Galaxia | Boss 依次点亮春夏秋冬四枚星核，每枚星核释放不同速度的弧形弹，最后四季星线同时收束成十字星河。 |
| 现实撕裂·RealityRupture | 现实撕裂 / Reality Rupture | Boss 在场地两侧撕开短暂裂隙，裂隙先吸引弹幕再喷出反向光刃，制造“进裂隙附近才安全”的反直觉走位。 |
