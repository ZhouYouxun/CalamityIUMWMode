# 白金星舰系武器攻击方式

> 目标：每一把武器都转写成一个 Boss 招式概念，只写方向、追踪、弹射、封路和阶段定位，不写具体数值。

---

## P1：星舰本体掉落

以星辉火焰、极光弹和爆破投掷为主。

| 招式 | 武器来源 | Boss 版设计 |
|------|----------|-------------|
| 星云鞭·Nebulash | 星云鞭 / Nebulash | Boss 甩出一条星云鞭影，鞭端会比鞭身晚半拍爆开，形成“先躲线、再躲点”的二段题。 |
| 极光烈焰枪·AuroraBlazer | 极光烈焰枪 / Aurora Blazer | Boss 向左右两侧交替扫射极光弹，弹幕颜色预示速度，蓝色慢封路，粉色快补位。 |
| 南翼星·AlulaAustralis | 南翼星 / Alula Australis | Boss 在自身两翼生成星羽，星羽先环绕蓄力，再从上下两层切过玩家。 |
| 北辉轰炸器·BorealisBomber | 北辉轰炸器 / Borealis Bomber | Boss 跳起后向下投放极光炸弹，炸弹落到玩家高度附近才横向爆裂成星片。 |
| 极光回旋镖·AuroradicalThrow | 极光回旋镖 / Auroradical Throw | Boss 投出极光回旋刃，去程追踪很弱，回程追踪增强，用来惩罚一直同向飞行。 |

## P2：星辉荒原解锁武器

由 Mantis、Atlas、Nova、Aries、Astralachnea、Astral Slime、Astraglomerate 等强化掉落补充。

| 招式 | 武器来源 | Boss 版设计 |
|------|----------|-------------|
| 星辉镰刀·AstralScythe | 星辉镰刀 / Astral Scythe | 场边出现两把星辉镰刀，沿弧线交叉收割，中点留有短暂安全窗。 |
| 泰坦臂·TitanArm | 泰坦臂 / Titan Arm | Boss 做一次重拳预警，随后拳影从玩家脚下向上击出，命中位置产生短范围冲击波。 |
| 恒星炮·StellarCannon | 恒星炮 / Stellar Cannon | Boss 架起恒星炮直线蓄能，预警线逐渐变亮，开火后弹体很快但只有一条清晰弹道。 |
| 恒星飞刀·StellarKnife | 恒星飞刀 / Stellar Knife | Boss 向上撒出一排飞刀，飞刀在空中停顿后按玩家当前速度方向斜落。 |
| 星幻蛛法杖·AstralachneaStaff | 星幻蛛法杖 / Astralachnea Staff | Boss 召出星蛛幻影，星蛛在场地边缘结出短线蛛网，蛛网只封路不追人。 |
| 遗弃史莱姆法杖·AbandonedSlime | 遗弃史莱姆法杖 / Abandoned Slime Staff | 场上生成小型星辉史莱姆核心，核心弹跳三次，每次落地喷出低矮星尘。 |
| 蜂巢荚·HivePod | 蜂巢荚 / Hive Pod | Boss 抛出星辉蜂巢，蜂巢裂开后释放小型蜂群，蜂群追踪弱但数量多，用来逼迫大位移。 |
