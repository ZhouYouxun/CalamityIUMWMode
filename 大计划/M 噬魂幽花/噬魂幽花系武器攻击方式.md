# 噬魂幽花系武器攻击方式

> 目标：每一把武器都转写成一个 Boss 招式概念，只写方向、追踪、弹射、封路和阶段定位，不写具体数值。

---

## P1：幽花本体掉落

幽灵、恐惧、钩索、火焰与召唤压制。

| 招式 | 武器来源 | Boss 版设计 |
|------|----------|-------------|
| 惊惧之刃·TerrorBlade | 惊惧之刃 / Terror Blade | 噬魂幽花把惊惧之刃改成预警斩击：斩线先在玩家前方亮起，短延迟后高速切过，末端再炸出一小圈碎刃。 |
| 女妖之钩·BansheeHook | 女妖之钩 / Banshee Hook | 噬魂幽花把女妖之钩改成有预警的近身斩线：先在玩家移动方向亮出刀痕，随后本体或幻影沿线横切。 |
| 魔鬼之焰·DaemonsFlame | 魔鬼之焰 / Daemon's Flame | 噬魂幽花投放魔鬼之焰爆裂核心，核心先慢慢漂到玩家附近，随后分裂成内外两圈火焰，外圈延迟半拍补位。 |
| 命运揭示·FatesReveal | 命运揭示 / Fate's Reveal | 噬魂幽花召出命运揭示法术核心，核心先漂移锁区，再释放有间隔的光束/魔弹。 |
| 幽魂面容·GhastlyVisage | 幽魂面容 / Ghastly Visage | 噬魂幽花召出幽魂面容副核心，副核心不追玩家，只周期性封住一片区域；Boss 本体趁空档做轻冲刺。 |
| 虚灵支配者·EtherealSubjugator | 虚灵支配者 / Ethereal Subjugator | 噬魂幽花短暂召唤虚灵支配者幻影协同攻击，召唤物持续封路，本体做低频冲刺。 |
| 食尸鬼钻掘者·GhoulishGouger | 食尸鬼钻掘者 / Ghoulish Gouger | 噬魂幽花把食尸鬼钻掘者改成预警斩击：斩线先在玩家前方亮起，短延迟后高速切过，末端再炸出一小圈碎刃。 |
