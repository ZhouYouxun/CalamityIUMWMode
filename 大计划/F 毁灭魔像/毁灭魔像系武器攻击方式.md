# 毁灭魔像系武器攻击方式

> 目标：每一把武器都转写成一个 Boss 招式概念，只写方向、追踪、弹射、封路和阶段定位，不写具体数值。

---

## P1：魔像本体掉落

肉块、火山、颅骨、尖刺与本体部件感。

| 招式 | 武器来源 | Boss 版设计 |
|------|----------|-------------|
| 终极裂肉刀·UltimusCleaver | 终极裂肉刀 / Ultimus Cleaver | Boss 举起巨刃砸向玩家预测点，砸落后左右各延伸一道短斩波。 |
| 领域毁灭者·RealmRavager | 领域毁灭者 / Realm Ravager | Boss 用身体撞开空间裂纹，裂纹从场地边缘向内推进，像一面缓慢压来的墙。 |
| 咯血·Hematemesis | 咯血 / Hematemesis | Boss 喷出抛物线血弹，血弹落点会二次溅射成低速红滴，压住地面路线。 |
| 尖刺岩杖·SpikecragStaff | 尖刺岩杖 / Spikecrag Staff | 地面或空中生成尖刺岩柱，先冒出碎石预警，再向玩家所在高度刺出。 |
| 颅骨粉碎者·CraniumSmasher | 颅骨粉碎者 / Cranium Smasher | Boss 放出巨大颅骨锤影，锤影横扫后短暂停在终点，再向 Boss 回收。 |
| 维苏威·Vesuvius | 维苏威 / Vesuvius | Boss 向上喷发火山弹，火山弹在空中裂成多枚下坠熔岩，形成雨幕。 |
| 血肉转向器·CorpusAvertor | 血肉转向器 / Corpus Avertor | Boss 分裂出血肉幻肢，幻肢从玩家两侧夹击，末端转向一次后消散。 |
| 血肉图腾·FleshTotem | 血肉图腾 / Flesh Totem | 场地生成血肉图腾，图腾周期发射环形小弹；Boss 本体此时放慢，让玩家决定是否优先清掉图腾。 |
