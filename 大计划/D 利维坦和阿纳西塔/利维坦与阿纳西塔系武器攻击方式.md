# 利维坦与阿纳西塔系武器攻击方式

> 目标：每一把武器都转写成一个 Boss 招式概念，只写方向、追踪、弹射、封路和阶段定位，不写具体数值。

---

## P1：海妖与巨兽双核

一个偏歌声/水魔法，一个偏巨兽冲撞/海浪火力。

| 招式 | 武器来源 | Boss 版设计 |
|------|----------|-------------|
| 碧潮·Greentide | 碧潮 / Greentide | Anahita 唱出一排绿色潮汐刃，潮汐刃先横向铺开，再按玩家移动方向逐片下压。 |
| 利维泰坦·Leviatitan | 利维泰坦 / Leviatitan | Leviathan 抬头后打出巨大水刃，水刃飞行很慢但体积大，擦过玩家后在身后炸成二段浪花。 |
| 阿纳西塔琶音·AnahitasArpeggio | 阿纳西塔琶音 / Anahita's Arpeggio | Anahita 在空中排出音符点，音符按琶音顺序发射水晶音波，节奏清楚但换向突然。 |
| 亚特兰蒂斯·Atlantis | 亚特兰蒂斯 / Atlantis | Boss 在玩家周围标出三条海蓝直线，短延迟后 Atlantis 光束贯穿整条线，迫使玩家在预警缝里横移。 |
| 胃液喷吐杖·GastricBelcher | 胃液喷吐杖 / Gastric Belcher Staff | Leviathan 召出小型胃囊，胃囊向玩家喷吐弧线酸液，酸液落点短暂残留，形成地面/空中双封锁。 |
| 白浪·Whitewater | 白浪 / Whitewater | Boss 让两股白浪从左右边界相向推进，白浪之间留出一个移动缺口，缺口会随着玩家高度微调。 |
| 利维坦之牙·LeviathanTeeth | 利维坦之牙 / Leviathan Teeth | Leviathan 甩出数枚牙齿飞镖，先扇形散开，随后带很弱追踪回切到玩家身后。 |
