# 风暴编织者系武器攻击方式

> 目标：每一把武器都转写成一个 Boss 招式概念，只写方向、追踪、弹射、封路和阶段定位，不写具体数值。

---

## P1：风暴编织者掉落

天空潮汐、风暴弓/枪和电能。

| 招式 | 武器来源 | Boss 版设计 |
|------|----------|-------------|
| 天潮龙骑枪·SkytideDragoon | 天潮龙骑枪 / Skytide Dragoon | 风暴编织者把天潮龙骑枪改成有预警的近身斩线：先在玩家移动方向亮出刀痕，随后本体或幻影沿线横切。 |
| 风暴·Storm | 风暴 / The Storm | 风暴编织者以风暴为核心放出电弧链，电弧先连接数个预警点，再按连接顺序跳跃放电，要求玩家记住线路。 |
| 伏特隆·Volterion | 伏特隆 / Volterion | 风暴编织者以伏特隆为核心放出电弧链，电弧先连接数个预警点，再按连接顺序跳跃放电，要求玩家记住线路。 |

| 特斯拉法杖·TeslaStaff | 特斯拉法杖 / Tesla Staff | 风暴编织者以特斯拉法杖为核心放出电弧链，电弧先连接数个预警点，再按连接顺序跳跃放电，要求玩家记住线路。 |
